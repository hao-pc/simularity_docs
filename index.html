223‑ФЗ: сравнение положений о закупках с эталоном

Работает на GitHub Pages (без сервера). Файлы остаются в браузере.

1) Эталон

Если критичный пункт отсутствует или сильно отличается — статус «изменения не внесены».
Настройки сравнения

2) Документы контрагентов

Имя контрагента берётся из имени файла (без расширения).
Для больших документов рекомендовано сначала сравнить 1–2 файла и проверить качество извлечения текста.

3) Результаты

Загрузите эталон и документы, затем нажмите «Сравнить».

Как выложить на GitHub Pages

  1. Создайте репозиторий и добавьте файлы index.html, styles.css, app.js.
  2. GitHub → Settings → Pages → Deploy from a branch → main / /root.
  3. Откройте выданный URL.

Важно: GitHub Pages — статический хостинг. Поэтому всё сравнение и генерация DOCX выполняются в браузере.
